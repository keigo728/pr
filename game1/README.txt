game.cpp : 球を動かし、障害物を避けた回数によってscoreを稼いでいく3Dゲーム

//操作方法//
  Aキー   : 球を右に移動
  Dキー 　: 球を左に移動
　Sキー　 : 球を中心に移動
SPACEキー : 球をジャンプさせる

! 注意 !
ゲーム中BGMが鳴ります。(音量注意)
球が障害物と衝突すると音が鳴ります。(音量注意)
球が障害物と衝突するとGAMEOVERです。画面が止まるので×ボタンでゲームを終了してください。


*プログラム内容*
第十二回課題を参考に作った。
ゲーム開始と同時にbgmを鳴らすように、bgm.wavをmain関数で流す。
当たり判定を障害物との距離として衝突時、bom.wavを流してfinflag = TRUEにしてtimer関数内のglutPostRedisplay()を止め、bgmも止める。
障害物は描画内で最大二種類までの描画で最初以外二種類描画されている。
障害物のパターンは15種類で、rand() % 15でランダムに出現、switch文でrand() % 15の値のパターンを描画している。
球はx,y軸で動かすことが可能で、障害物はz軸で近づいてくる。
障害物は既定のZ座標を超えた瞬間、初期座標Zの値に戻し、その時scoreを+1していく仕組み。
ジャンプは放物線を描くようにY座標を動かすことでジャンプしているように見せている。
